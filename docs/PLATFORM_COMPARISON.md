# Platform Comparison

Side-by-side comparison of all 12 supported platforms.

## Content Specs

| Platform | Char Limit | Media | Tags/Hashtags | Link Handling |
|----------|-----------|-------|---------------|---------------|
| Reddit | 40K body, 300 title | Images, videos, polls | Flair (1 per post) | URL posts or text |
| LinkedIn | 3,000 | Images, videos, docs | 3-5 hashtags | In comments (better reach) |
| Facebook | 63,206 | Images, videos, stories | Hashtags (limited use) | Rich preview cards |
| Threads | 500 | Up to 10 images | Topic tags | Preview cards |
| X (Twitter) | 280 | 4 images or 1 video | 1-2 hashtags max | t.co shortened |
| Product Hunt | Varies | Product screenshots | Topic tags | Product URL required |
| Hacker News | Title only | None | None | URL or text (not both) |
| Dev.to | No limit | Cover image + inline | Up to 4 tags | Inline markdown links |
| IndieHackers | No limit | Images | Group selection | Inline links |
| Bluesky | 300 | Up to 4 images | None (no hashtags!) | Rich cards |
| Substack | No limit | Images, embeds | None | Inline links |
| Medium | No limit | Images, embeds | Up to 5 tags | Inline links |

## Audience Profile

| Platform | Primary Audience | Best For |
|----------|-----------------|----------|
| Reddit | Tech communities, niche interests | Community engagement, feedback |
| LinkedIn | Professionals, B2B | Thought leadership, networking |
| Facebook | General public, 25-55+ | Community groups, local reach |
| Threads | Social media users, 18-35 | Quick thoughts, engagement |
| X (Twitter) | Tech, media, real-time | Hot takes, threads, networking |
| Product Hunt | Tech early adopters, founders | Product launches |
| Hacker News | Senior developers, founders | Technical content, Show HN |
| Dev.to | Developers, beginners to senior | Tutorials, career advice |
| IndieHackers | Bootstrapped founders, makers | Build-in-public, milestones |
| Bluesky | Tech/design/journalism | Conversational, early adopter |
| Substack | Newsletter readers, niche audiences | Long-form, subscriber growth |
| Medium | General readers, SEO traffic | Articles, SEO visibility |
